# Сборка
Для сборки в системе должен быть установлен Maven. Открываем терминал в рабочей папке проекта и выполняем команду:
```
mvn package
```

# Запуск
```
java -jar target/test-task-0.0.1.jar
```

# БД
Для демонстрации используется h2 in memory db. Можно использовать другую СУБД, поменяв соответствующие настройки в файле **application.properties** и добавив зависимости драйверов БД в **pom.xml**.
